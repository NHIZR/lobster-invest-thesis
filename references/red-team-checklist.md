# Red Team Checklist

Run before finalizing any report. Every item must be addressed. Skipping items produces weak research.

---

## Pre-Publish Checks

### Thesis Quality
- [ ] **Strongest bear case articulated?** Not a straw man — the real objection that keeps you up at night.
- [ ] **Kill conditions specific and measurable?** "If things get worse" is not a kill condition. "$X price for Y days" is.
- [ ] **What evidence would prove me wrong?** Name the specific data point, metric, or event.
- [ ] **Who's on the other side of this trade?** Smart money selling to you, or dumb money panicking?

### Bias Detection
- [ ] **Am I anchoring on recent news?** Would I hold the same view if last week's headline hadn't happened?
- [ ] **Am I anchoring on my entry price?** Sunk cost has no place in forward-looking analysis.
- [ ] **Confirmation bias check:** Did I seek out counter-arguments, or only evidence supporting my thesis?
- [ ] **Narrative seduction:** Is the story compelling but the numbers mediocre?

### Report Completeness
- [ ] **Devil's Advocate section written?** All three perspectives (Burry, Graham, Druckenmiller) represented with distinct attacks.
- [ ] **Technical Snapshot included?** (If price data available)
- [ ] **Sentiment Pulse included?**
- [ ] **All claims have source links?** No naked assertions.
- [ ] **Valuation uses 2+ methods?**
- [ ] **Position sizing specified?** Entry, target, stop, size, rationale.

### Source Quality
- [ ] **3+ Tier 1 sources?** Primary data, not secondary commentary.
- [ ] **No fabricated quotes?** If you don't have the exact quote, paraphrase and cite.
- [ ] **Data freshness?** Numbers from this quarter, not last year.
- [ ] **Cross-referenced?** Key claims verified from 2+ independent sources.

### Decision Readiness
- [ ] **IRR exceeds hurdle rate?** Stock ≥15%, Short ≥20%, Crypto ≥50%.
- [ ] **Entry price specified?** Not "around current levels" — an actual number.
- [ ] **Timeline defined?** When should the thesis play out?
- [ ] **Edge match checked?** Step 0 completed, edge (or lack thereof) acknowledged.

---

## Scoring Gate

Self-score 1-10 on each dimension:

| Dimension | Score | Notes |
|-----------|-------|-------|
| Thesis clarity | /10 | |
| Evidence quality | /10 | |
| Valuation rigor | /10 | |
| Risk framework | /10 | |
| Decision readiness | /10 | |
| **Average** | **/10** | |

**Below 7.5 average:** Revise before publishing. Identify the weakest dimension and strengthen it.

**Below 6.0 average:** The report isn't ready. Either spend more time or downgrade to "Watch" with a note on what's missing.
